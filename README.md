Road Accident Severity Prediction using Explainable Machine Learning with Spatio-Temporal Risk Analysis

Project Overview

Road traffic accidents remain one of the leading causes of fatalities and serious injuries worldwide. Predicting accident severity can help traffic authorities, emergency responders, and policymakers take proactive measures to reduce casualties and improve road safety.

This project develops an Explainable Machine Learning (XML) framework to predict road accident severity using historical accident data. The framework incorporates robust data preprocessing, compares multiple machine learning models, provides interpretable predictions through feature importance analysis, and performs spatio-temporal risk analysis to identify accident hotspots and high-risk time periods.


---

Objectives

Predict road accident severity using multiple machine learning algorithms.

Compare the performance of different classification models.

Handle missing and noisy accident data through robust preprocessing.

Explain model predictions using Explainable AI .

Identify spatial and temporal accident trends and hotspots.

Determine the most influential factors contributing to accident severity.
